# Архитектура Progyx

## Принципы

- модульная fullstack-архитектура
- разделение ролей и политик доступа
- изолированные API-секции по доменам
- seed-контент для быстрого демо и хакатонной защиты

## Backend

- `auth.py` — регистрация, вход, refresh, logout, /me
- `student.py` — dashboard, roadmap, уроки, тесты, задания, классы
- `teacher.py` — классы, задания, просмотр сдач
- `admin.py` — обзор системы, управление модулями, суперадмин-функции

## Frontend

- Landing и auth flow
- Dashboard по текущему пользователю
- Roadmap с GSAP-анимацией
- Lesson player с теорией + Monaco + quiz
- Разделы teacher/admin/superadmin для демонстрации RBAC

## Расширение

- новые модули добавляются сидированием или через admin API
- новые роли можно подключить через `UserRole` и policy-checks
- новый тип задания добавляется через `Task.task_type` + новый компонент на фронтенде
- родительский кабинет можно вынести в отдельный blueprint и отдельный dashboard
